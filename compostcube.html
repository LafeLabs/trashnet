in the dirt web we build cubes out of cardboard trash which can carry dirt. We grow plants and fungi in dirt. We also use the cubes to carry art supplies like box cutters, elmer's glue and markers, and GEOMETRON rulers to make more cubes, making the cubes fully self-replicating. Cubes can contain cardboard media which can be hyperlinks, symbols, icons, geometric art, or functional elements like phone stands. Cubes can also contain physical digital media like USB thumb drives. Cubes are used as electronics enclosures and structural elements of robots. When cubes break down over time, they are sterilized in pressure cookers and put in sealed bags with mushroom grow medium. Arrays of electrodes are inserted into grow medium, and the electrodes are connected to analog in and digital out of an Arduino or some related Adafruit product. The Arduino is controlled by python code on a trash server which the community can co-edit and evolve over time into a system which applies pulses to the mycelium and recieves responses. The goal of this evolution will be to make the mycelium behave as a programmable circuit which can run Geometron and Arduino-like code. If they can run Arduino-like code, they can replace the Arduino which controls the pulses, making a system where the mycelium programs the mycelium, but is connected over USB to a server which random people from around the world can hack on.
