TRASH MAGIC WEB

WEB EDITOR FORK

The purpose of Trash Magic is to create a civilization of free things entirely built from trash. We will build things from trash, share them freely, and share how to build them with others so that they can replicate. We call this replication from person to person of the knowhow and desire to make things "magic".

The TRASH NET is the global and local network of trash. The network of web pages built on servers from the trash is called the TRASH WEB. We use the TRASH WEB to build web pages representing places, people and trash. We use these to create TRASH MAGIC QUESTS, which are actions people can take to build towards full TRASH MAGIC. We go out into public spaces and connect physical trash with the TRASH MAGIC WEB, and create QUESTS for people to follow to transform all of the trash in the world into self-replicating free things.
